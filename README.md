# 🚂 Railway Automation Testing Project

### 📝 Project Overview
This project focuses on building an automated testing framework for the **Safe Railway** system to optimize the testing process and ensure software quality. The automation suite covers critical business flows, including **Login, Book Ticket, and User Registration**.

### 🛠 Technologies Used
* **Language:** Java
* **Framework:** Selenium WebDriver
* **Design Pattern:** Page Object Model (POM) 
* **Test Management:** TestNG
* **Build Tool:** Maven

### 📊 Test Execution Results
* **Total Test Cases:** 16
* **Execution Status:** 100% Automated scripts executed successfully.
* **Bug Identification:** The scripts accurately detected **9 real-world bugs** (Functional & UI issues) within the system.
* **Evidence Collection:** Leveraging automation results to generate detailed bug reports, including manual screenshots as evidence for each failed scenario.
